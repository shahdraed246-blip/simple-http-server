Computer Networks

Computer networks are built on a layered structure that enables communication between different devices through various networking infrastructures. The protocol stack provides an organized approach to network communication, where each layer has specific responsibilities and interacts with adjacent layers. This modular design allows flexibility, scalability, and easier troubleshooting.

Main Layers of the Internet Protocol Model:

  1. Application Layer
  2. Transport Layer
  3. Network Layer
  4. Link Layer
  5. Physical Layer

Main Network Components

  • Routers and Switches (Core)
  • Access Networks (WiFi, DSL, Cable)
  • End Systems (Clients and Servers)
  • Protocols (TCP/IP, HTTP, FTP)
  • Transmission Media (Fiber, Copper, Wireless)
