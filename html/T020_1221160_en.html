Request a File

Enter the file name you want to access from our server. The system will check if the file exists and if you have permission to access it.

Instructions:

  • Available files: Try requesting files like "notes.txt" or "report.pdf" or any other file name
  • Private files: Files containing "private" in their name are forbidden
  • File not found: Non-existing files will return a 404 error
  • Success: Correct files will be served with a 200 OK response

Sample Files for Testing:

notes.txt private_doc.txt report.pdf
